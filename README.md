# YouTube RAG Expert

YouTube RAG Expert is a Retrieval-Augmented Generation (RAG) application that turns any YouTube video into an interactive learning experience.  
From a single YouTube URL, you can extract transcripts, generate structured notes, ask intelligent questions, and download everything as a PDF.
